3) Write a query that selects all orders for amounts greater than any for the customers in London.

mysql> SELECT *
    -> FROM orders
    -> WHERE Amt > ANY (
    ->     SELECT Amt
    ->     FROM orders o
    ->     JOIN customers c ON o.Cnum = c.Cnum
